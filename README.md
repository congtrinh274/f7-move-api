# Rescue Car API
## Introduction
A Back end Application using NodeJS Express, SocketIO,.. builds API for vehicle rescue application. 

## Technology
* NodeJS Express
* BcryptJS
* Sequelize
* Socket.io
*..
## Features
* Customer API
  - login
  - register
  - get car brand
  - get car type
  - get color car
  - send rescue
* Partner API
  - login
  - register
  - get services
